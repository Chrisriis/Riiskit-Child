# Riiskit Child

This is the child theme for Riiskit, a developer focused parent theme that's based on Underscore, HTML5 Boilerplate and SMACSS.

Riiskit: https://github.com/Chrisriis/riiskit

## Installation
Install Riiskit first.

Then you need to have installed both _npm_ and _gulp_. After that, you simply locate (cd) the folder with your CLI and write:

* _npm install_
* _gulp_

Just wait a while for the packages to install and voila!
